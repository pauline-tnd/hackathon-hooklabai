# Photo Generation

Photo generation is available **after you reveal a caption**. It’s triggered from the revealed caption screen via **Generate image**.

Each click generates **one** image. There’s no option list.

### Flow

1. Reveal a caption (costs **1 credit**).
2. Tap **Generate image**.
3. (Optional) tweak the prompt or style settings.
4. Wait for the image to render.
5. The image is shown on the result screen.

Generating the image costs **1 credit**.

### After generation

After generation, you can:

* Download / save the photo.
* Attach it to a Base post.

{% hint style="info" %}
After you pay, you should get the full-quality asset. Avoid charging again for the same asset.
{% endhint %}
